# FabricatedForgeEventBus
Forge's EventBus, but working on fabric/quilt.
This mod is indev!

</p>

## How to use?

```groovy

repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    modImplementation 'com.github.TexTrueStudio:FabricatedForgeEventBus:fabric-SNAPSHOT'
}

```
